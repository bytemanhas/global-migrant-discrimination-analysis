# 🌍 Global Migrant Discrimination Analysis (Power BI)

## 📌 Project Overview
This project analyzes global data on migrant discrimination using Power BI to identify regional patterns, trends over time, and key factors influencing incidents. The dashboard provides insights that can support data-driven policy decisions and awareness initiatives.

---

## 🎯 Objective
The objective of this project is to:
- Analyze discrimination trends across different regions
- Identify high-risk areas
- Understand patterns over time
- Provide insights for policy and awareness

---

## 🛠️ Tools & Technologies
- Power BI
- Microsoft Excel (Data Preparation)
- DAX (Data Analysis Expressions)

---

## 📂 Dataset Description
The dataset contains global records related to migrant discrimination with the following fields:
- Country / Region
- Year
- Number of Incidents
- Type of Discrimination
- Severity Level
- Demographic Information

---

## 📊 Dashboard Features

### KPI Cards:
- Total Incidents
- Average Incidents per Region
- Highest Affected Region

### Visualizations:
- Incidents by Region (Bar Chart)
- Trend Analysis Over Time (Line Chart)
- Type of Discrimination Distribution (Pie/Bar Chart)
- Severity Analysis (Column Chart)

### Filters / Slicers:
- Region
- Year
- Type of Discrimination

---

## 🔍 Key Insights
- Certain regions consistently report higher incidents
- Trends vary over time depending on socio-economic factors
- Some types of discrimination are more prevalent than others
- Incident levels differ significantly across regions

---

## 💡 Recommendations
- Strengthen monitoring in high-risk regions
- Improve reporting systems for accurate data collection
- Promote awareness programs to reduce discrimination
- Support policy interventions based on regional insights

---

## 📌 Conclusion
This dashboard demonstrates how Power BI can be used to analyze global social issues and provide structured insights to support informed decision-making.
